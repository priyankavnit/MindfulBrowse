Installation Instructions

  1. Click the download button above to get the extension ZIP file
  2. Extract the ZIP file to a folder on your computer
  3. Open Chrome and navigate to chrome://extensions
  4. Enable Developer mode using the toggle in the top-right corner
  5. Click Load unpacked button
  6. Select the folder where you extracted the extension
  7. The Mindful Browse extension should now appear in your toolbar!
Note: This is a development version of the extension. For production use, the extension would be published to the Chrome Web Store for easier installation.
